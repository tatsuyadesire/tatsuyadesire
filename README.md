# Olá! Me chamo Lou! ⋆⭒˚.⋆

˙⋆✮ **`Analista de Dados`** ✮⋆˙

Sou discente de Ciência de Dados e atuo na área de Análise de Dados e Automatizações de Tarefas. Trabalho principalmente com Python e Visual Basic for Applications (VBA), desenvolvendo soluções para otimizar processos, organizar informações e apoiar a tomada de decisões por meio de dados. Tenho experiência na criação de controles automatizados em Excel, desenvolvimento de dashboards e construção de ferramentas que tornam atividades operacionais mais eficientes. Minha trajetória na indústria contribuiu para uma visão prática sobre processos, melhoria contínua e resolução de problemas. Também possuo conhecimentos em métodos ágeis, Scrum, comunicação e organização de projetos. Busco constantemente aprimorar minhas habilidades em programação, análise de dados e automação, aplicando criatividade, foco e adaptabilidade para desenvolver soluções úteis e funcionais.
